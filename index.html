Comando Git Exemplo de uso Operação
Config git config --global user.name | user.email Configura o usuário e e-mail
help git help <comandos> Obtém ajuda sobre um comando específico
Init git Init Inicializar um repositório em uma pasta
status git status Verificar a situação (status) do repositório
add <file> git add . | git add <arquivo> Adiciona um ou mais arquivos ao stagin area
commit git commit -m "mensagem" Cria a versão (commita) o repositório
reset git reset <arquivo> Volta do estado preparado para o modificado
restore git restore <arquivo> Desfaz uma alteração em um arquivo modificado
clean git clean -fd Desfaz a criação (apaga) um arquivo criado
branch git branch -M "main" Mudar o nome da branch principal
code code . | code <arquivos> Abre o VScode ou na pasta ou um arquivo
